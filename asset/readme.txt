AIGI-Holmes asset/ 资源目录
============================

【文件夹作用】
本目录存放 AIGI-Holmes 系统使用的静态资源文件，包括应用图标和相关图片资源。

【各文件说明】

  app.ico                   应用程序图标（ICO 格式）
                            用于：
                            - Windows 可执行文件（.exe）的程序图标
                            - 桌面快捷方式图标
                            - 开始菜单图标
                            - Inno Setup 安装程序图标
                            建议包含多尺寸：16×16、32×32、48×48、256×256

  teasor.png                项目宣传图/效果预览图
                            展示系统界面和检测效果的截图或示意图

  datapipeline.png          数据处理流程图
                            系统数据流转和处理管道的示意图

  cr_datapipeline.pdf       数据处理流程文档（PDF格式）
                            详细描述系统数据处理流水线的技术文档
