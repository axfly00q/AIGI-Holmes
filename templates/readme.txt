AIGI-Holmes templates/ 模板目录
================================

【文件夹作用】
本目录存放 AIGI-Holmes 的 Jinja2 HTML 模板文件，
由 FastAPI 后端渲染后返回给用户浏览器。

【各文件说明】

  index.html                前端主页模板
                            AIGI-Holmes 检测系统的主界面，包含：
                            - 图片上传区域（拖拽/点击/粘贴 URL）
                            - 单图、URL批量、文件批量三种检测模式切换
                            - 检测结果展示（判定结论/置信度/Grad-CAM热力图）
                            - CLIP 内容分类标签和图文一致性评分
                            - 豆包 AI 流式分析对话框
                            - 报告导出按钮（PDF/Excel）
                            - 用户登录/注册入口
                            对应路由：GET /

  admin.html                管理后台模板
                            面向 admin 角色用户的后台管理页面，包含：
                            - 系统统计仪表盘（用户数/检测数/造假率趋势图）
                            - 用户列表管理（搜索/角色筛选/角色变更）
                            - 检测记录查询（来源搜索/标签筛选/分页）
                            - 用户反馈记录列表和训练集集成操作
                            对应路由：GET /api/admin/page
