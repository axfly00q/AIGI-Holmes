AIGI-Holmes 新闻图片 AI 生成检测系统
================================================================================

【项目简介】

AIGI-Holmes 是一款企业级 AI 生成图片检测系统，用于识别和溯源新闻媒体中的
人工智能生成图片。通过深度学习模型、内容分析和多维度验证，帮助记者、编辑
和审核人员快速甄别假图和虚假信息。

核心功能：
  • 单张/批量图片检测（AI 生成判定 + 置信度）
  • URL 采集检测（自动爬取新闻页面所有图片并检测）
  • 图文一致性分析（基于 CLIP 零样本分类）
  • 多轮 AI 分析（豆包 AI 流式解释检测结果）
  • 报告导出（PDF/Excel 格式）
  • 用户认证与管理后台


================================================================================
【快速开始】- 三选一
================================================================================

✅ 最简单: 双击 dist\AIGI-Holmes\AIGI-Holmes.exe

  1. 解压或复制 dist\AIGI-Holmes\ 文件夹到本地
  2. 双击 AIGI-Holmes.exe
  3. 等待 10-20 秒，程序自动打开桌面窗口
  4. 上传图片进行检测


✅ 安装版: 双击 dist\AIGI-Holmes-Setup.exe

  1. 按照安装向导完成安装
  2. 自动创建桌面快捷方式
  3. 点击快捷方式启动程序


✅ 源代码运行:

  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements-app.txt
  python desktop_launcher.py

【关键文件一览】
================================================================================

desktop_launcher.py
  → 应用启动器，启动 FastAPI + PyWebView 窗口
  → 直接运行这个文件启动应用

backend/main.py
  → FastAPI 应用主入口
  → uvicorn backend.main:app 启动

detect.py
  → 核心检测模块，调用 ResNet50 进行推理

finetuned_fake_real_resnet50.pth
  → AI 检测模型参数（80 MB）
  → 必须存在于项目根目录

.env.example
  → 环境变量配置示例
  → 复制为 .env 并填写配置

AIGI_Holmes.spec
  → PyInstaller 打包配置
  → pyinstaller AIGI_Holmes.spec 生成 EXE

installer.iss
  → Inno Setup 安装脚本
  → iscc installer.iss 生成 Setup.exe

README.md
  → 项目总体说明（技术栈、API、架构）

USAGE.md
  → 详细使用教程（功能说明、故障排查）

readme.txt
  → 本文件（文件夹内所有文件说明）

【技术栈简览】
================================================================================

后端       : FastAPI + Uvicorn
检测       : PyTorch + ResNet50（微调）
分类       : CLIP ViT-B/32
数据库     : SQLite（默认）/ PostgreSQL（生产）
缓存       : Redis（可选）
认证       : JWT + Bcrypt
报告       : PDF / Excel
桌面       : PyWebView + Edge WebView2
打包       : PyInstaller + Inno Setup

【常见问题】
================================================================================

Q: 如何修改为只使用 FastAPI 后端？
A: 已完全支持！启动命令：
   uvicorn backend.main:app --host 127.0.0.1 --port 7860
   或在浏览器访问：http://127.0.0.1:7860

Q: 怎样查看 API 文档？
A: 启动应用后访问 http://127.0.0.1:7860/docs（Swagger UI）

Q: 可以部署到服务器吗？
A: 完全可以，支持 Docker、纯 Python 等多种部署方式

Q: 程序启动很慢？
A: 首次启动会加载模型（10-20 秒），后续更快

Q: 检测准确率是多少？
A: 约 95%（基于 ResNet50 微调模型）
