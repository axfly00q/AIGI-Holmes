# 🚀 AIGI-Holmes 完整使用指南

## 📦 安装方式（选择其一）

### ✨ 方式 1：绿色版（最简单，推荐）

**适用于**: 所有用户

**步骤**:
1. 获取 `dist/AIGI-Holmes/` 文件夹
   - 直接从项目复制 `dist/AIGI-Holmes/` 整个文件夹
   - 或将其解压到任意位置

2. **双击** `AIGI-Holmes.exe` 启动
   ```
   AIGI-Holmes.exe
   ```

3. 等待 10-20 秒加载模型

4. **程序会自动**：
   - ① 打开桌面窗口（如果系统有 Edge WebView2）
   - 或② 打开浏览器访问 `http://127.0.0.1:7860`
   - 或③ 在控制台输出访问地址

5. 🎉 开始使用！

**包含文件**:
```
AIGI-Holmes/
├── AIGI-Holmes.exe          ← 主程序（双击这个）
├── python.exe               ← Python 运行时
├── _internal/               ← 所有依赖库（无需理会）
├── templates/               ← 前端模板
├── static/                  ← 前端资源
└── finetuned_fake_real_resnet50.pth  ← AI 模型
```

**特点**:
- ✅ 无需 Python 环境
- ✅ 无需额外安装
- ✅ 无需配置数据库
- ✅ 开箱即用


### 🔧 方式 2：Windows 安装程序（推荐用于分发）

**适用于**: 需要正式安装的用户

**前置条件**:
- 需要安装 Inno Setup 6（用于生成安装程序）

**步骤**:
1. （仅开发者）生成安装程序：
   ```bash
   iscc installer.iss
   ```
   生成文件: `dist/AIGI-Holmes-Setup.exe`

2. 用户获取 `AIGI-Holmes-Setup.exe`

3. **双击** `AIGI-Holmes-Setup.exe` 安装

4. 按照向导完成安装（选择安装路径）

5. 安装完成后：
   - ✅ 桌面出现快捷方式
   - ✅ 开始菜单出现快捷方式
   - ✅ 支持卸载

6. 双击快捷方式启动 AIGI-Holmes


### 🛠️ 方式 3：源代码运行（开发者）

**前置条件**:
- Python 3.9+ 环境
- 所有依赖已安装

**步骤**:
```bash
# 进入项目目录
cd AIGI-Holmes-main

# 方式 A：用桌面窗口启动
python desktop_launcher.py

# 方式 B：纯后端 API（用浏览器访问）
uvicorn backend.main:app --host 127.0.0.1 --port 7860 --reload
```

**访问应用**:
- 自动打开窗口，或
- 浏览器访问：`http://127.0.0.1:7860`

## 🎯 功能演示

### 功能 1: 单张图片检测

1. 打开首页
2. 点击"上传图片"或直接拖拽图片
3. 点击"开始检测"
4. 等待 1-2 秒获得结果
5. 查看：
   - ✅ 判定结果（FAKE / REAL）
   - ✅ 置信度百分比
   - ✅ Grad-CAM 热力图
   - ✅ 内容标签分类

### 功能 2: 批量检测

1. 需要管理员或审核员权限
2. 上传多张图片（ZIP 或单个文件）
3. 实时查看检测进度
4. 导出报告（PDF 或 Excel）

### 功能 3: URL 采集检测

1. 粘贴新闻页面 URL
2. 自动采集页面所有图片
3. 自动进行批量检测
4. 查看图文一致性分析

### 功能 4: 用户认证

1. 点击"注册"创建账户
2. 输入用户名和密码
3. 登录后可查看历史记录
4. 管理员可在后台管理用户权限

### 功能 5: 报告导出

1. 检测完成后，点击"导出报告"
2. 选择格式：
   - 📄 **PDF** - 正式报告，含水印

1. 以 admin 身份登录
2. 访问 `/admin` 页面
3. 查看统计数据、用户管理、反馈管理等

1. 检测完成后，点击"AI 分析"
2. Shift + Enter 支持多行输入
3. 提问关于检测结果的问题
4. AI 实时流式返回分析结果

## 🔍 故障排查

### 问题 1: 程序无法启动
**症状**: 双击 EXE 后没有反应

**解决方案**:
```
1. 等待 20-30 秒（首次需加载模型）
2. 打开 %TEMP%\AIGI-Holmes.log 查看日志
3. 检查是否有错误信息
4. 如果仍无反应，用浏览器访问：http://127.0.0.1:7860
```

### 问题 2: 端口 7860 已被占用
**症状**: 提示"端口已使用"

**解决方案**:
```bash
# 查看占用的进程
netstat -ano | findstr :7860

# 杀死该进程（PID 为上面查到的进程号）
taskkill /PID <PID> /F

# 重新启动程序
```

### 问题 3: 无法看到桌面窗口
**症状**: 程序启动了但没有打开窗口

**解决方案**:
```
1. 这是正常的（缺少 Edge WebView2）
2. 用浏览器手动访问：http://127.0.0.1:7860
3. 所有功能仍然可用
4. 或安装 Edge WebView2：
   https://developer.microsoft.com/zh-cn/microsoft-edge/webview2/
```

### 问题 4: 检测很慢
**症状**: 每张图片需要 5-10 秒

**解决方案**:
```
1. 首次启动加载模型（正常）
2. 使用 CPU 模式较慢（5-10 秒）
3. 建议安装 CUDA 加速到 <1 秒
4. 或等待后续优化版本
```

### 问题 5: 忘记数据库密码
**症状**: 无法登录

**解决方案**:
```
1. 删除 aigi_holmes.db 文件
2. 重启程序（会自动创建新数据库）
3. 用新账户重新注册
```
