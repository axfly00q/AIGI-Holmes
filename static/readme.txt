AIGI-Holmes static/ 前端静态资源目录
======================================

【文件夹作用】
本目录存放 AIGI-Holmes Web 前端的所有静态资源文件（样式表和 JavaScript 脚本），
由 FastAPI 通过 /static 路由提供访问。

【各文件说明】

  css/style.css             全局样式表
                            定义系统整体视觉风格，包括：
                            - 页面布局与响应式设计
                            - 检测结果卡片（AI生成/真实照片 判定样式）
                            - 上传区域、进度条、热力图展示区
                            - 用户登录注册界面
                            - 管理后台数据表格与统计图表样式

  js/app.js                 前端主逻辑脚本
                            实现所有前端交互功能，包括：
                            - 图片上传与拖拽检测（单图/批量）
                            - URL 页面图片抓取与批量检测
                            - WebSocket 实时进度接收与展示
                            - 检测结果渲染（热力图、置信度、CLIP 标签）
                            - 豆包 AI 流式分析文字输出（SSE）
                            - 用户注册/登录/JWT 令牌管理
                            - 报告导出（PDF/Excel）请求
                            - 管理后台数据查询与展示
