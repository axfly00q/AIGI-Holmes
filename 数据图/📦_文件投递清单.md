  - 模块化设计（每个图为独立函数）
  - 完整的中文标签和说明
  - 高清 DPI=300，适合打印
  - 可自定义颜色和样式
- **使用方式**:
  ```bash
  python 算法数据流图.py
  ```
- **输出**: 5个 PNG 文件 + 执行日志
- **依赖**: matplotlib, numpy, Pillow
- **执行时间**: ~10 秒

#### 2️⃣ `使用数据流图示例.py` (~400行)
- **功能**: 交互式图表查看和索引生成工具
- **菜单功能**:
  1. 显示单张图表 (可缩放、平移)

  3. 提取特定阶段细节 (双栏对比显示)
  4. 生成 Markdown 索引 (自动生成 `数据流图索引.md`)
  5. 退出
- **使用方式**:
  ```bash
  python 使用数据流图示例.py
  ```
- **依赖**: matplotlib, Pillow
- **交互方式**: 命令行菜单
