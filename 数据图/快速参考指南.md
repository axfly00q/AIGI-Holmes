### 如何优化性能？

| 瓶颈 | 优化方案 | 对应图表 |
|------|---------|---------|

| 大量连接 | 连接池复用 | 04-第3层优化建议 |
| 内存溢出 | 流式处理 | 04-第3层的Queue |

## 各阶段数据大小估算

### 单图检测
```
输入  : JPG 图片           ~ 200KB-5MB
中间  : Tensor 224×224 × 3  ~ 0.6MB (内存)
输出  : JSON 结果          ~ 50KB (含Base64热力图)
```

### 批量检测 (100张)
```
输入  : 图片列表           ~ 20-100MB
缓冲  : 内存 Queue         ~ 2GB (配置相关)
输出  : JSON 结果列表      ~ 5MB
```

### 报告生成
```
PDF   : 单页报告           ~ 500KB-2MB
Excel : 完整数据表         ~ 200KB
JSON  : 结构化输出         ~ 100KB
```
